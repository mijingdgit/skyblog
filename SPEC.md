# 3D 炫彩博客 - 规格文档

## 1. 项目概述

- **项目名称**: SkyBlue 3D 博客
- **项目类型**: 个人博客网站 (Vue3 + TypeScript + Element Plus + Three.js)
- **核心功能**: 展示博客文章的个人网站，带有炫酷的3D视觉效果
- **目标用户**: 博客访客、技术爱好者

## 2. UI/UX 规格

### 布局结构

- **导航栏**: 固定顶部，高度60px，包含Logo和导航链接
- **Hero区域**: 全屏3D粒子背景+标题+副标题，占据首屏
- **博客卡片区域**: 瀑布流/网格布局展示文章卡片
- **关于区域**: 个人简介+技能展示
- **页脚**: 联系方式+版权信息

### 响应式断点
- 移动端: < 768px
- 平板: 768px - 1024px
- 桌面: > 1024px

### 视觉设计

#### 颜色方案 (主色调: SkyBlue)
- **主色 (Primary)**: `#00BFFF` (DeepSkyBlue)
- **次要色 (Secondary)**: `#1E90FF` (DodgerBlue)
- **强调色 (Accent)**: `#00FFFF` (Cyan)
- **渐变1**: `linear-gradient(135deg, #00BFFF 0%, #00FFFF 50%, #1E90FF 100%)`
- **渐变2**: `linear-gradient(45deg, #0ea5e9, #06b6d4, #14b8a6)`
- **背景色**: `#0a0a1a` (深色星空背景)
- **卡片背景**: `rgba(255, 255, 255, 0.05)` (毛玻璃效果)
- **文字色**: `#ffffff` (白色)
- **次要文字**: `rgba(255, 255, 255, 0.7)`

#### 字体
- **标题字体**: "Orbitron", "Noto Sans SC", sans-serif (科技感)
- **正文字体**: "Noto Sans SC", "Microsoft YaHei", sans-serif
- **标题大小**:
  - H1: 3.5rem
  - H2: 2.5rem
  - H3: 1.5rem
- **正文大小**: 1rem (16px)

#### 间距系统
- 基础间距: 8px
- 小间距: 16px
- 中间距: 24px
- 大间距: 48px
- 区块间距: 80px

#### 视觉效果
- **卡片**: 毛玻璃效果 `backdrop-filter: blur(10px)`, 边框发光
- **按钮**: 渐变背景 + 发光阴影 + hover放大
- **文字**: 发光效果 (text-shadow)
- **3D背景**: Three.js粒子星空 + 旋转几何体

### 组件列表

1. **NavBar**: 固定导航栏
2. **HeroSection**: 3D全屏欢迎区
3. **BlogCard**: 博客文章卡片
4. **BlogGrid**: 文章网格容器
5. **AboutSection**: 关于我区域
6. **SkillTag**: 技能标签
7. **Footer**: 页脚

### 组件状态

- **按钮Hover**: 放大1.05 + 发光增强
- **卡片Hover**: 边框发光 + 轻微上浮
- **导航链接Hover**: 颜色渐变 + 下划线动画

## 3. 功能规格

### 3D 场景功能
- 粒子星空背景 (500+粒子)
- 旋转的彩色几何体 (立方体、八面体、二十面体)
- 鼠标交互: 粒子轻微跟随鼠标
- 自动缓慢旋转动画

### 博客功能
- 展示6篇示例博客文章
- 分类标签筛选
- 卡片Hover显示摘要
- 点击跳转到详情(模拟)

### 导航功能
- 首页、关于、文章分类
- 平滑滚动
- 移动端汉堡菜单
